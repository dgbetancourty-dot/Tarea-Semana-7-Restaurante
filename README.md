## Descripción

Este proyecto fue desarrollado en Python utilizando Programación Orientada a Objetos.

El sistema permite registrar productos y clientes mediante un menú interactivo. También permite listar y buscar la información registrada.

Durante el desarrollo se aplicaron conceptos como:

- Clases y objetos.
- Encapsulamiento mediante `@property` y `@setter`.
- Uso de `@dataclass`.
- Arquitectura modular.
- Separación de modelos, servicios y programa principal.
